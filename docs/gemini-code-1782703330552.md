## 💰 2. Tracks y Distribución de Premios
[cite_start]El hackathon cuenta con una bolsa total de premios dividida en tres verticales competitivas independientes[cite: 11]:

1. [cite_start]**Track 1: Multiverse Agents ($2,000 USD)** * *Premio al mejor caso de uso Multi-Agente + Multimodal*[cite: 11, 44].
   * [cite_start]Evalúa la coordinación efectiva entre múltiples agentes de IA y el uso significativo de capacidades de texto e imagen de Gemma 4[cite: 45].
   * [cite_start]Se premia el impacto de la velocidad de Cerebras e implementaciones creativas fuera de la caja orientadas al mundo real (IA física, robótica, sistemas embebidos, IoT, etc.)[cite: 46].

2. [cite_start]**Track 2: People's Choice ($2,000 USD)** * *Premio a la mayor cantidad de impresiones en redes sociales*[cite: 11, 47].
   * [cite_start]Se evalúa el alcance orgánico (sin promoción pagada) y el nivel de interacción de la comunidad (likes, comentarios, reposts) que genere el video de la demo compartido en la plataforma X (Twitter)[cite: 21, 48, 51].

3. [cite_start]**Track 3: Enterprise Impact ($1,000 USD)** * *Premio al mejor caso de uso empresarial*[cite: 11, 53].
   * [cite_start]Enfocado en resolver retos empresariales reales de misión crítica (búsqueda multimodal, RAG corporativo, ciberseguridad, respuesta a incidentes, soporte técnico avanzado)[cite: 55, 56, 57].
   * [cite_start]Evalúa la escalabilidad, arquitectura robusta y preparación del código para entornos de producción[cite: 58, 59].

## ⚙️ 3. Especificaciones de la API de Cerebras

[cite_start]Para este evento, la infraestructura de Cerebras Cloud otorga capacidades premium y límites extendidos bajo el formato compatible con la API de OpenAI[cite: 63, 108]:

* [cite_start]**Capacidad de Rate Limits Elevada:** 100 RPM (Llamadas por minuto) y 100K TPM (Tokens por minuto) por participante[cite: 63, 73].
* [cite_start]**Ventana de Contexto Expandida:** Capacidad máxima incrementada hasta **65K MSL / 32K MCL** para el manejo de documentos o flujos extensos[cite: 136].
* [cite_start]**Soporte Multimodal:** Entrada de imágenes y salida exclusiva en texto[cite: 107]. [cite_start]Las imágenes deben enviarse incrustadas obligatoriamente como **Base64 data URIs** dentro del parámetro `image_url` (las URLs de imágenes hospedadas en internet no están soportadas en esta versión preview)[cite: 109, 110].
* [cite_start]**Structured Outputs y Tool Calling:** Soporte completo para llamadas a funciones (*Tool Calling*) y salidas estructuradas forzadas mediante el parámetro `strict: true` para restringir la respuesta del modelo a un JSON Schema estricto[cite: 111, 112].
* [cite_start]**Modo de Razonamiento (Thinking Mode):** El razonamiento avanzado está apagado por defecto[cite: 95, 114]. [cite_start]Para activarlo, se debe configurar explícitamente el parámetro `reasoning_effort` con los valores `low`, `medium` o `high`[cite: 115, 116]. Para mantenerlo apagado, se usa `none`[cite: 115].
* [cite_start]**Métricas de Rendimiento:** Cada respuesta (*completion*) inyecta un objeto `time_info` que detalla los tiempos exactos de procesamiento (TTFT, tokens por segundo y latencia de punta a punta) ideales para demostrar la velocidad en el proyecto[cite: 117, 118, 120, 122, 123].

## 📝 4. Reglas de Desarrollo y Requisitos de Entrega

* [cite_start]**Boilerplate y Scaffolding:** Está permitido utilizar código base preexistente, plantillas, boilerplate y frameworks de desarrollo anteriores[cite: 83, 84]. [cite_start]No obstante, el núcleo de la aplicación y la lógica funcional del proyecto deben ser construidos estrictamente dentro de las 24 horas del hackathon, teniendo a Gemma 4 corriendo en Cerebras como el motor central[cite: 84].
* [cite_start]**Línea Base Comparativa:** Está permitido llamar a un segundo proveedor de modelos (como Gemini) únicamente para realizar comparaciones visuales lado a lado del rendimiento y latencia dentro de la demo[cite: 132, 133].
* [cite_start]**Requisitos del Video Demo:** * Duración máxima estricta de **60 segundos**[cite: 38].
  * [cite_start]Debe demostrar explícitamente la ventaja en velocidad de la inferencia ultra-rápida de Cerebras (se recomienda una comparativa contra una GPU tradicional)[cite: 39, 40].
  * [cite_start]**Privacidad Obligatoria:** Durante la grabación del escritorio no se debe mostrar información sensible o confidencial (API keys, notificaciones, correos electrónicos, pestañas del navegador o credenciales)[cite: 42].
